Οδηγίες εγκατάστασης ολόκληρου του συστήματος:

1. Σύνδεση των στοιχείων όπως φαίνεται στην αναφορά.

2.Εγκατάσταση του Arduino IDE -> https://support.arduino.cc/hc/en-us/articles/360019833020-Download-and-install-Arduino-IDE.

3. Εγκατάσταση του Processing 4 IDE->https://processing.org/de/download/ .

4. Άνοιγμα αρχείου 'Arduinocode' απο τον αντίστοιχο υποφάκελο και φόρτωση προγράμματος στο Arduino.

5. Άνοιγμα αρχείου 'Processingcode' και ενεργοποίηση του πατόντας "Run".

*Προσοχή , ελέγξτε αν ταιριάζει η πύλη του Arduino (πχ "COM3") με αυτή που δέχεται δεδομένα το Pocessing.
**Μην έχετε ανοιχτό το serial monitor στο arduino IDE διότι δεν θα λειτουργεί το Processing.
***Μην διαγράψτε το αρχείο "sketch" διότι είναι σημαντικό για την εκτέλεση του "Processingcode".
****Αν θέλετε να μεταφέρετε τα αρχεία τότε το "Arduinocode" πρέπει να βρίσκεται στον υποφάκελο του, και το "Processingcode" πάντα μαζί με το sketch.
